== Installation ==

1. Upload `wp-bashorg.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Cange settings in widgets menu 

== Frequently Asked Questions ==

Q: How change widget title? 
R: Enter in manage themes->widgets, move widget Bashorg in sidebar and change title.
